Puntos 1 y 2

Se generó un json con data aleatoria con 500,000 registros, incluyendo los campos de nombre, email y password. Esta información fue publicada en las APIs:

  • http://127.0.0.1:5000/password_lengths
  • http://127.0.0.1:5000/email_domains

Que proporcionan datos sobre la longitud de las contraseñas y el número de dominios de correo electrónico diferentes, respectivamente. Posteriormente, se consumieron estas APIs utilizando JavaScript y se representaron gráficamente utilizando la biblioteca Plotly. Los resultados se presentan en la página principal index.html.
